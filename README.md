# Arcade

Play arcade games and change graphics library or game dynamically. Developed in C++.

![alt text]

## Add your own graphics library and game

```
foo
```

## Compile

You need to install :

* [SFML](https://www.sfml-dev.org/index-fr.php)
* [libcaca](http://caca.zoy.org/wiki/libcaca)

Then :

```
./make
```

## Run

```
./arcade
```
